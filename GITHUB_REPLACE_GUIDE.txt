새 앱 기능
- 간편 단계(low/medium/high) 입력 유지
- exact W 세밀 입력 추가
- CFD 학습범위 내 interpolation / 범위 밖 warning
- 목표온도 0.1°C 단위
- 54개 HVAC 후보 가상시험
- 현재 추정 hotspot 위치와 추천 후 예상 변화
- Top-5 hotspot 해소 여부
- 현재/추천 후 동일 색상범위 온도장 비교
- Before/After 전체 노드 CSV 다운로드
- Hotspot 변화 CSV 다운로드
